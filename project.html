Naudotojo patirties (UX) duomenų automatinis rinkimas, apibendrinimas ir panaudojimas

Automatic Collection, Summarization, and Utilization of User Experience (UX) Data

Anotacija (LT): Projektas siekia pagerinti programinės įrangos aplikacijų naudojamumą sistemingai vertinant ir renkant naudojamumo statistiką iš naudotojo sąsajų. Šis duomenimis grindžiamas požiūris padės programuotojams nustatyti, kurios programos savybės reikalauja tobulinimo, taip pat prioritetus ir kriterijus, kuriuos reikia pasirinkti. Pavyzdžiui, projektas koncentruosis į dažniausiai atliekamų naudotojų veiksmų tobulinimą, tuo pačiu užtikrinant, kad mažiau paplitę veiksmai išlaikytų minimaliai priimtiną našumą.
Be to, projektas stebės įvairių naudotojų veiksmų vykdymo laikus, analizuodamas jų trukmę atsižvelgiant į užduočių imtis ir sudėtingumą. Šis stebėjimas leis iš anksto informuoti naudotojus apie galimus veikimo sulėtėjimus, siūlant galimas priemones, tokias kaip sudėtingų užduočių skaidymas į paprastesnes, siekiant pagerinti bendrą efektyvumą.

Abstract (EN): The project aims to improve the usability of software applications by systematically evaluating and collecting usability statistics from user interfaces. This data-driven approach will help developers identify which features of the application require improvement, as well as the priorities and criteria to focus on. For example, the project will concentrate on enhancing the most frequently performed user actions while ensuring that less common actions maintain a minimally acceptable level of performance.
Additionally, the project will monitor the execution times of various user actions, analyzing their duration in relation to task samples and complexity. This monitoring will enable proactive notifications to users regarding any slowdown in performance, suggesting potential actions such as breaking down complex tasks into simpler steps to improve overall efficiency.

Raktažodžiai: Naudotojo patirtis (UX), Naudojamumo testavimas, Duomenų rinkimas
